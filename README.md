A deep learning-based system for classifying skin diseases including Acne, Eczema, and Psoriasis using medical images. The project leverages VGG16 and ResNet-50 architectures with transfer learning, data augmentation, and class balancing techniques for high accuracy.

Features:

Upload skin images for classification

Classifies into Acne, Eczema, or Psoriasis

Uses fine-tuned CNN models for prediction

Technologies: Python, TensorFlow/Keras, OpenCV,  Matplotlib
